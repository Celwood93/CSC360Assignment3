type make into the command line to run the make file

type ./diskinfo disk.IMA to see the info of the disk
type ./disklist disk.IMA to see a list of the files on the disk
type ./diskget disk.IMA ans1.pdf to get the file ans1.pdf off the disk
type ./diskput disk.IMA test.txt to put the file test.txt onto the disk
type ./diskget disk.IMA test.txt to get the file TEST.TXT off the disk, note it is identical to test.txt.
